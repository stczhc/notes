为什么量子场论是现在这个样子? 因为 (除了允许有无限粒子类型的弦论) 它是调解量子力学原理和狭义相对论原理的唯一方式. 在这本书中路径积分的引入较后, 作者的理由是虽然它是到目前为止快速导出费曼规则的最好方式, 但是它模糊了隐藏在这些计算下的量子力学原因. 狄拉克方程负能解解释为负能态海洋, 对应的, 实验观测到了正电子. 但是对自旋为整数的粒子, 没有泡利不相容原理, 因此无法使用这个解释. 但是实验也观测到了对应的反粒子, 但是狄拉克这个理论无法解释. 因此没有理由认为对于费米子就应该这样解释. 实际上, 量子场论的发展使反粒子的空穴解释不再必要.
